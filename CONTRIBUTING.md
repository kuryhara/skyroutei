# Contributing

## Branch naming

- `feature/<short-name>`
- `fix/<short-name>`
- `docs/<short-name>`

## Before committing

```powershell
.\scripts\check_project.ps1
```

## Commit examples

- `Add dynamic map legend`
- `Fix resource mission destinations`
- `Improve Huai'an scenario stages`

## Pull requests

Describe:

1. what changed;
2. why it changed;
3. how it was tested;
4. screenshots for visual changes.

Do not add API keys, passwords, `.osm.pbf` files, generated graphs or personal paths.
