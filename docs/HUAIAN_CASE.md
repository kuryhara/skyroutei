# Huai'an chlorine-leak live simulation

The application includes a scenario based on the March 29, 2005 liquid-chlorine leak on the Beijing–Shanghai Expressway in Huai'an, Jiangsu.

In the interface, the scenario is presented as an unfolding live incident. The agent receives information progressively, ranks vulnerable targets, assigns resources to explicit missions, generates street-following routes, checks conflicts and waits for human confirmation.

Historical outcome figures and uncertain reconstruction assumptions should remain clearly distinguished in the project documentation and presentation materials.
